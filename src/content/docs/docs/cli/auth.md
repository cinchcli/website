---
title: cinch auth
description: Authentication and device pairing commands.
---

`cinch auth` manages authentication and device pairing.

## Subcommands

### `cinch auth pair [TOKEN]`

Pair this machine with a relay server using a pairing token.

```bash
cinch auth pair abc123
```

### `cinch auth status`

Check current authentication status.

```bash
cinch auth status
```

### `cinch auth logout`

Remove stored credentials.

```bash
cinch auth logout
```
